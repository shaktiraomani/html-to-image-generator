# HTML to Image Converter

A sleek, web-based developer tool to convert HTML and CSS directly into high-quality downloadable images. Perfect for sharing code snippets, UI designs, and quick mockups. Everything runs locally in your browser—no backend required.

## Features

- ⚡️ **Live HTML & CSS Editor**: Write HTML & CSS in separate tabs and see changes instantly.
- 👁️ **Real-time Preview**: See exactly what your image will look like before downloading.
- 📱 **Responsive Widths**: Test and capture your designs in Mobile (375px), Desktop (800px), or Auto widths.
- 🖼️ **Multiple Formats**: Export your creations safely to PNG or JPEG format.
- 🎨 **Modern Interface**: Designed with a sleek, developer-friendly dark editor and clean preview environment.

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Core Engine**: `html-to-image`

## Getting Started

Follow these steps to set up the project locally:

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/your-username/html-to-image-converter.git
   cd html-to-image-converter
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   The app will typically be available at `http://localhost:3000` (or `http://localhost:5173` depending on your Vite config).

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.
